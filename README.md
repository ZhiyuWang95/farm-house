# farm-house

Personal interview preparation workspace.

## Purpose

This repo organizes interview prep by **company → position → interview type**. Each position folder contains four areas:

- **coding** — LeetCode-style problems with Python solutions
- **design** — system design notes and trade-offs
- **behavior** — behavioral interview prep (STAR stories, etc.)
- **domain_knowledge** — role-specific technical concepts and references

### Coding layout

Each problem lives in its own folder, named `lc-{id}-{slug}/`:

```
google/sr-swe/coding/lc-1-two-sum/
  solution.py
  notes.md
```

### Other areas

Design, behavior, and domain knowledge use topic folders with a `notes.md`:

```
google/sr-swe/design/rate-limiter/notes.md
```
